Over Mij

Hallo, ik ben Husseen, een tweedejaars student Software Developer bij het Techniek College Rotterdam. Ik ben 17 jaar oud en kom uit Ceel Barde, Somalië.

In mijn vrije tijd speel ik graag voetbal met vrienden, game ik en kijk ik films en series.

Ik heb ervaring met HTML, CSS, JavaScript en PHP. Onlangs heb ik het project Chirpify afgerond, waarvoor ik HTML, CSS, JavaScript en PHP heb gebruikt.

Ook heb ik als project een website gemaakt voor het Techniek College Rotterdam, waarbij ik HTML, CSS en JavaScript heb toegepast.

Ik vind het leuk om te leren en mijn vaardigheden te verbeteren terwijl ik aan deze projecten werk.
